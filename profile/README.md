## 关于我们

我们是一群热爱技术与创新的开发者，专注于利用 AI 能力构建实用的桌面应用、Web 工具和命令行程序。从图像处理到代码可视化，从函数绘图到系统优化，我们探索 AI 与工程结合的无限可能。

## 项目一览

### 桌面应用

| 项目 | 描述 | 技术栈 |
|:-----|:-----|:-------|
| [**Tools**](https://github.com/ChaseToDream/Tools) | 插件化桌面工具箱，支持本地/npm 插件安装管理 | Electron + Vue 3 + SQLite |
| [**Code-K**](https://github.com/ChaseToDream/Code-K) | 代码交易所 — 把 Git 仓库当股市可视化，每个文件是"股票"，每次 commit 是 K 线 | Electron + React 19 + lightweight-charts |
| [**StarCalculator**](https://github.com/ChaseToDream/StarCalculator) | 跨平台桌面计算器，支持数学计算、数据换算与工程计算 | Tauri 2 + Vue 3 + Rust |

### Web 应用

| 项目 | 描述 | 技术栈 |
|:-----|:-----|:-------|
| [**TRAE-Forum**](https://github.com/ChaseToDream/TRAE-Forum) | TRAE 社区帖子展示，三视图 + 分类筛选 + 关键词搜索 + 每 2 小时自动更新 | JavaScript + Python + GitHub Actions |
| [**AIGCwiki**](https://github.com/ChaseToDream/AIGCwiki) | AI 绘画作品展示平台 | Next.js + TypeScript |
| [**Function-Visualization**](https://github.com/ChaseToDream/Function-Visualization) | 数学函数可视化工具，实时绘图 + 多函数对比 + 数据导出 | React 18 + Zustand + mathjs |
| [**Copybook**](https://github.com/ChaseToDream/Copybook) | 字帖生成器，水墨纸韵风格的中国书法美学 | TypeScript + Vite |
| [**IMGconfusion**](https://github.com/ChaseToDream/IMGconfusion) | 图片混淆与还原工具，保护图片隐私 | TypeScript + Tailwind CSS |
| [**watermark**](https://github.com/ChaseToDream/watermark) | 图片水印消除工具，智能检测 + 手动涂选 + 批量处理 | React + FastAPI + OpenCV |

### 命令行工具

| 项目 | 描述 | 技术栈 |
|:-----|:-----|:-------|
| [**CleanCLI**](https://github.com/ChaseToDream/CleanCLI) | Windows 系统垃圾深度清理，40+ 垃圾类型 + 残留检测 + 模拟模式 | Python（零依赖） |
